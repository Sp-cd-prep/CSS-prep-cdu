Combinators

combinators combine the selectors to provide them a useful relationship and the position od content in the document 

    1. Descendant selector(space) - select all the child elements that are present in side one specific element(parent).

    2. Child selector(>) - it will affect elements under a direct parent. 
    3. Adjacent sibling selector(+) - it is used to select an element that is directly after a specific element. or it will target the direct sibling.
    4. General sibling selector(~) - it will target all the elements which is comes after a specific element.
